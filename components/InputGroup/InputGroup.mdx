# InputGroup

O componente `InputGroup` permite adicionar prefixos, sufixos ou ícones a um campo de input, criando uma unidade visual coesa. É ideal para formulários que necessitam de contexto adicional, como unidades de medida, domínios de site ou ícones de identificação.
