# Math Text (LaTeX)

O componente `Math` renderiza expressões matemáticas formatadas usando a biblioteca `react-katex`, que utiliza a sintaxe **LaTeX**.

> ℹ️ Observação: Para que a renderização funcione, o CSS do KaTeX (`katex/dist/katex.min.css`) deve ser importado globalmente na aplicação ou no `preview.js` do Storybook.
