# Input

O componente `Input` é uma abstração sobre o `TextField` do Material-UI, projetado para ser versátil e fácil de usar em formulários. Ele suporta vários tipos, estados e adornos.

## Uso Básico

Este é o input padrão, com a variante "outlined".

<Canvas>
  <Story of={InputStories.Default} />
</Canvas>

## Adornos (Adornments)

Adornos podem ser usados para adicionar prefixos, sufixos ou ícones a um campo de texto, o que é útil para exibir unidades de medida ou dar dicas visuais.
