# Modal

O componente `Modal` (ou `Dialog`) exibe conteúdo em uma camada sobre o aplicativo, desativando a interação com a página principal até que uma ação seja tomada ou o modal seja fechado.

## Uso Básico

Este é um exemplo de um modal padrão com um texto simples.

<Canvas>
  <Story of={ModalStories.Default} />
</Canvas>

## Com Conteúdo Longo

Quando o conteúdo excede a altura do modal, uma barra de rolagem é adicionada automaticamente.

<Canvas>
  <Story of={ModalStories.WithLongContent} />
</Canvas>

## Com Ações no Rodapé

É comum adicionar botões de ação no rodapé do modal. Você pode fazer isso utilizando o componente `DialogActions` do Material-UI.

  <Story of={ModalStories.WithActions} />
</Canvas>
